# JLPT 학습 카드 웹 앱

JLPT 시험 대비를 위해 문제를 풀면서  
문장 속 일본어 표현과 한자를 즉시 확인할 수 있도록 설계된  
**브라우저 기반 일본어 학습 웹 애플리케이션**입니다.

별도의 설치 없이 웹 브라우저에서 바로 사용할 수 있으며,  
문제 풀이 흐름을 유지한 상태에서 **어휘 · 한자 · 발음**을 함께 학습하는 것을 목표로 합니다.

---

## 🚀 주요 기능

### 📘 JLPT 문제 카드 학습

- JLPT 형식의 객관식 문제를 카드 방식으로 제공합니다.
- 문제를 선택하면 즉시 정답 여부를 확인할 수 있습니다.
- 정답 해설과 함께 오답이 왜 틀렸는지에 대한 설명이 제공됩니다.
- 문제는 순차적으로 풀거나, 원하는 번호로 바로 이동할 수 있습니다.

---

### 🈶 문장 속 한자 즉시 확인

- 문제 문장과 해설에 포함된 한자를 클릭하면  
  해당 한자의 정보를 바로 확인할 수 있습니다.
- 제공 정보:
  - 읽는 법(음독 · 훈독)
  - 한글 뜻
  - 획순(쓰기 순서)
- 별도의 사전 검색 없이 학습 흐름이 이어집니다.

---

### 🔊 문장 발음 듣기 (TTS)

- 문제 문장에 대해 일본어 발음을 음성으로 들을 수 있습니다.
- 발음 듣기 버튼을 통해 문장을 자연스럽게 따라 읽으며 학습할 수 있습니다.
- 발음 확인은 문제 풀이 흐름을 방해하지 않도록  
  필요할 때만 실행할 수 있습니다.
- 브라우저 환경에 따라 음성 품질이나 지원 여부에 차이가 있을 수 있습니다.

---

### 📝 해설 중심 학습 구조

- 정답 여부만 제시하지 않고,
  - 왜 이 선택지가 맞는지
  - 다른 선택지는 왜 틀렸는지
  를 중심으로 설명합니다.
- 동일한 문법이나 표현이 사용된 추가 예문을 통해
  실제 사용 맥락을 함께 학습할 수 있습니다.

---

### 💾 학습 진행 상태 관리

- 현재 문제 위치와 전체 진행 상황을 확인할 수 있습니다.
- 맞힌 문제와 틀린 문제가 구분되어 표시됩니다.
- 학습 진행 상태는 자동으로 저장되어,
  페이지를 닫아도 이어서 학습할 수 있습니다.

---

## 💡 학습 편의 기능

- PC, 태블릿, 스마트폰 등 다양한 환경에서 사용 가능합니다.
- 키보드와 터치 입력을 모두 지원합니다.
- 한자 정보 창은 필요할 때만 열리고,
  닫으면 다시 문제 화면으로 바로 돌아옵니다.
- 화면 구성은 장시간 학습에도 부담이 적도록 단순하게 설계되었습니다.

---

## 📖 기본 사용 방법

1. 페이지에 접속합니다.
2. 문제를 하나 선택해 풉니다.
3. 정답을 선택하면 결과와 해설을 확인합니다.
4. 모르는 한자가 있으면 클릭해 정보를 확인합니다.
5. 발음이 궁금한 경우 발음 듣기 버튼을 눌러 확인합니다.
6. 다음 문제로 이동하며 학습을 이어갑니다.

## 📄 라이선스 및 출처

이 프로젝트의 애플리케이션 코드와 사용자 인터페이스는  
개인 학습 및 비상업적 목적에 한해 자유롭게 사용할 수 있습니다.

---

### 🔹 한자 획순 표시 UI (kaki-jun 웹컴포넌트) 출처

본 프로젝트의 **한자 획순 표시 모달 기능**은  
브라우저에서 한자의 획순(Stroke Order)을 시각적으로 보여주기 위한  
학습 보조용 사용자 인터페이스입니다.

이 기능은 다음의 오픈소스 프로젝트들을 기반으로 구성되어 있습니다.

- **kaki-jun (Web Component)**  
  https://github.com/code4fukui/kaki-jun  
  KanjiVG 기반 SVG 파일을 브라우저에서 표시하고  
  획순 애니메이션을 재생할 수 있도록 제공하는  
  `<kaki-jun>` 커스텀 태그(웹컴포넌트) 구현 프로젝트입니다.

- **kanjivganimate**  
  https://github.com/nihongodera/kanjivganimate  
  KanjiVG SVG 데이터를 기반으로  
  획순 애니메이션을 구현하기 위한 공개 애니메이션 라이브러리입니다.

- **KanjiVG Project**  
  https://github.com/KanjiVG/kanjivg  
  한자의 획순 정보를 SVG 형식으로 제공하는  
  공개 데이터 프로젝트입니다.

본 프로젝트는 위 구성요소를 **학습용 표시(UI)** 목적으로만 사용하며,  
획순 데이터나 애니메이션 엔진을 직접 제작하거나 소유하지 않습니다.  
각 구성요소의 저작권과 라이선스는 해당 원 프로젝트에 귀속됩니다.

---

### 🔹 한자 읽기 및 한글 뜻 정보 출처

본 프로젝트에서 제공되는 한자 읽기(음독 · 훈독),  
한글 뜻 및 관련 정보는 다음의 공개 데이터 프로젝트 및 API를 기반으로 합니다.

- **kanjiapi.dev**  
  https://kanjiapi.dev  
  KanjiVG, KANJIDIC 등 공개 사전 데이터를 종합하여  
  한자 정보를 API 형태로 제공하는 공개 프로젝트입니다.

- **usejun/kanji**  
  https://github.com/usejun/kanji  
  JLPT 학습을 위해 한자 읽기 및 의미 정보를  
  JSON 형태로 정리한 공개 데이터 프로젝트입니다.

위 프로젝트들은 한자 데이터를 직접 제작한 것이 아니라,  
공개 사전 및 획순 데이터를 학습 목적으로 가공·제공합니다.

---

본 프로젝트는:
- 한자 획순 데이터 및 의미 데이터를 직접 제작하지 않으며
- 학습 편의를 위해 이를 **표시하는 인터페이스(UI)**만 제공합니다.

외부 데이터 및 API의 라이선스 조건은  
각 원 출처 프로젝트의 안내를 따르며,  
상업적 이용 또는 2차 배포 시에는 반드시 개별 라이선스를 확인해야 합니다.

---

본 프로젝트는 학습 목적의 데모로 제공되며,  
외부 데이터 및 API의 정확성·완전성·지속성에 대한 보증을 포함하지 않습니다.
